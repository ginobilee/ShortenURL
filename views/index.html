FreeCodeCamp Challenge: Shorten URL service

This is a page to get shorten URL.It is used to:

  • Pass a url as a parameter,Then you will get a shortened URL instead.
  • You can use the shortened URL to get to the original page.
  • You can click here to get all existing shortened URLs.

Example Usage:

  • https://shorten-url-masonlee.c9users.io/add/http://www.sohu.com
  • https://shorten-url-masonlee.c9users.io/add/http://www.baidu.com
  • https://shorten-url-masonlee.c9users.io/list

You can click these shortened urls to check where it redirect you.
